una volta completato il backend, eseguire il comando seguente per creare in automatico tutti i servizi crud in angular per comunicare 

npx openapi-generator-cli generate -g typescript-angular -i http://localhost:8000/api/v1/openapi.json -o frontend/src/app/api


Aggiunta librerie: modifica backend/pyproject.toml oppure:
uv add nome-pacchetto
per rimouvere:
uv remove nome-pacchetto
(in automatico aggiorna le dipendenze)


se faccio una nuova tabella devo creare il modello in models e poi importarlo in init su __all__
per aggiornare il db:
uv run alembic revision --autogenerate -m "add appointment table"
uv run alembic upgrade head

per far partire il backend:
uv run uvicorn app.main:app --reload --port 8000
